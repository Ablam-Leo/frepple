=======
Segment
=======

Segments represent a collection of item + location combinations that matches the criteria of the segment.

They can be used for filtering purpose to only display on the screen the item-locations belonging to a given segment.
They can also be used in combination with one or multiple business rules to define a common inventory
policy to all item-locations belonging to a segment.
